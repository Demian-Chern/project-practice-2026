# Описание проекта «МосПолиФизикс»

## Общая информация
**Название проекта:** МосПолиФизикс  
**Автор:** Чернявский Демьян Алексеевич  
**Тип проекта:** Научно-популярный + технический

## Актуальность
С 2026 года физика становится обязательным вступительным экзаменом для большинства инженерных направлений Московского Политеха. При этом у большинства школьников наблюдается низкая мотивация и слабая подготовка по предмету. В университете отсутствует единый современный проект, который бы популяризировал физику и показывал её прикладное значение в инженерии.

## Цель проекта
Повысить интерес школьников к физике и помочь им успешно подготовиться к вступительным экзаменам через современные цифровые форматы.

## Задачи
- Создание качественного научно-популярного контента
- Разработка Telegram-бота для взаимодействия с аудиторией
- Создание интерактивного сайта проекта
- Участие в Днях открытых дверей МосПолитеха
- Реализация простого блокчейна как демонстрационного технического проекта

## Техническая часть (выполненная в рамках практики)
- Разработан полнофункциональный **простой блокчейн** на Python + Flask
- Создан клиент для генерации кошельков и подписания транзакций
- Реализован веб-интерфейс для майнинга и просмотра цепочки блоков

## Технологический стек
- Python, Flask
- HTML, CSS, JavaScript, Bootstrap
- Git + GitVerse
- Библиотеки: Crypto, hashlib, Flask-CORS

## Результаты
- Рабочий прототип блокчейна с майнингом, транзакциями и консенсусом
- Полноценный сайт практики
- Полная документация проекта
