## Основные знания, которые получил

- Архитектура **Transformer** и принцип работы современных LLM.
- Техники эффективного промпт-инжиниринга (Chain of Thought, Few-shot, Role prompting, Tree of Thoughts).
- Как правильно использовать LLM для генерации кода, документации и текстов.
- Основные риски и этические вопросы использования ИИ (галлюцинации, bias, авторские права, конфиденциальность).
- Способы проверки и улучшения качества ответов моделей.
