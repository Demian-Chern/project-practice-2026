## 2. Архитектура системы

Система состоит из двух основных компонентов:

1. **Blockchain Node (Майнер)** — `blockchain.py`
   - Хранение цепочки блоков
   - Обработка и валидация транзакций
   - Реализация алгоритма Proof of Work
   - REST API для взаимодействия

2. **Blockchain Client (Кошелёк)** — `blockchain_client.py`
   - Генерация криптографических ключей (RSA)
   - Создание и подписание транзакций
   - Удобный веб-интерфейс

## 3. Реализованные механизмы

- **Криптографическая подпись транзакций** (RSA + PKCS1_v1_5 + SHA)
- **Proof of Work** (поиск nonce с заданным количеством ведущих нулей)
- **Цепочка блоков** с хешированием предыдущего блока (SHA-256)
- **Генерация кошельков** (публичный/приватный ключ)
- **REST API** для создания транзакций, майнинга и просмотра цепочки
- **Веб-интерфейсы** для удобной демонстрации работы
